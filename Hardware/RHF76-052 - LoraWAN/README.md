## **Especificaciones Tecnicas**
- Alto presupuesto de enlace de 160dB. -140dBm de sensibilidad y 19dBm de potencia de salida.
- Banda dual 868 / 915MHz
- 14dBm @ 868MHz / 915MHz
- Interfaz amigable
- SPI / USART / I2C / USB;
- 2 × ADC;
- 10 × GPIOs
- Soporte del protocolo LoRaWAN
